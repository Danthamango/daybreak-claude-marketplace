# Daybreak brand kit (condensed)

Applied to HTML reports.

## Colors

| Name | Hex | Use in reports |
|---|---|---|
| Sunlight | `#FECF00` | Attention callouts, stuck-patient chips, CTA accents, positive highlights |
| Deep Sleep | `#221D35` | Primary text, headers, borders |
| Pillow | `#FFFFFF` | Main background |
| Sky | `#DDE4ED` | Informational callouts, section dividers |
| Linen | `#F6ECE1` | Soft section backgrounds, footer |

## Typography

- **Halyard Display** (primary): headers, body, metrics. Web fallback: `Inter, system-ui, -apple-system, "Segoe UI", sans-serif`
- **Nantes Light** (serif, accent): used sparingly for emphasis words in hero headlines. Fallback: `Georgia, "Source Serif Pro", serif`

## Tone

- "Radically easy"
- "On your side"
- "Expert driven"
- "Better days start with better sleep"

## Applied choices in the template

- Report title uses Halyard-ish Inter, weight 600, color Deep Sleep
- Accent words in exec header use Nantes-ish Georgia italic, Sunlight underline
- Stuck-patient rows use a 3px left border in Sunlight
- Section headers use Deep Sleep at weight 600, uppercase, 1px letter-spacing
- Info callouts use Sky background, Deep Sleep text
- Footer uses Linen background
